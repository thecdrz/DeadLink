Overview

DeadLink is a Discord-first companion for 7 Days to Die. It provides narrative AI reports, analytics, and Blood Moon intelligence while keeping operations safe.

Install

Clonegit clone https://github.com/thecdrz/DeadLink.git
Installnpm install
Configurecopy config.example.json to config.json and edit
Runnpm start

Configuration

KeyDescription
DISCORD_TOKENYour bot token
DISCORD_CHANNELChannel ID for messages
TELNET_IPServer IP
TELNET_PORTTelnet port
TELNET_PASSWORDPassword from serverconfig.xml

AI Reports

Context-aware activity summaries with location, time, health, and Blood Moon cues.

Analytics

Trends, sessions, MVP, distance metrics, and optional image charts.

Blood Moon

Predictive alerts with cooldowns, Discord notices, and in-game broadcast.

Security

Scoped telnet queue with timeouts, retries, reconnects, and admin-only controls.
