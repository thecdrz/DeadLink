Quick Start

  1. Create a .env file (or docker run -e) with your Discord and server details (see README for all variables): 
    DISCORD_TOKEN=...
TELNET_HOST=server.example.com
TELNET_PORT=8081
TELNET_PASSWORD=...
  2. Pull the container: 
    docker pull ghcr.io/thecdrz/deadlink:latest
  3. Run it: 
    docker run --name deadlink --restart unless-stopped \
  --env-file .env \
  -v %cd%\data:/app/data \
  -d ghcr.io/thecdrz/deadlink:latest
    On Linux/macOS, replace %cd% with $(pwd).

Prefer environment variables over command-line flags. No local Node.js required when using Docker.
