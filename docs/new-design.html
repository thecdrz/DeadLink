Powerful Features

DeadLink transforms your server management experience with these core capabilities, designed to make administration easier and player engagement deeper.

Player & Server Trends

Track and visualize online player counts, server restarts, and latency metrics. Spot potential issues early and understand your community's gaming patterns.

Daily Activity Reports

Receive comprehensive daily summaries of all server activity directly to your Discord. Track logins, deaths, chat messages, and other important in-game events.

Horde Night Alerts

Never be caught unprepared again. DeadLink provides smart, rate-limited blood moon warnings with intelligent cool-downs to prevent notification spam.

Safe Server Controls

Manage your server through secure, permission-based commands for restarts, messaging, and moderation. No more dealing with raw telnet commands and their potential risks.

Quick Installation

DeadLink is designed to be easy to deploy. Just run the Docker container with your server details, and you're ready to go.

bash
docker run --name deadlink \\
-e DISCORD_TOKEN=... \\
-e TELNET_HOST=... \\
-e TELNET_PASS=... \\
ghcr.io/thecdrz/deadlink:latest
DeadLink v2.7.0 initializing... [INFO] Connected to telnet server [INFO] Connected to Discord [SUCCESS] DeadLink is now monitoring your server!
